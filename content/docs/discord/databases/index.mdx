Using the [CLI to generate your project](/discord/start), you can select a database preset.

In most presets, the chosen database integration is in the `database` folder within `src`.

The available presets are:

- MongoDB (mongoose) - [See the guide](/discord/databases/mongodb)
- Firestore (typesaurus) [See the guide](/discord/databases/firestore)
- Firestore (firelord) [See the guide](/discord/databases/firestore)
- QuickDB (better-sqlite3)
