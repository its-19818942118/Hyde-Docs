Usando a [CLI para gerar seu projeto](/discord/start), é possível selecionar uma predefinição de banco de dados.

Na maioria das predefinições a integração com o banco de dados escolhida fica na pasta `database` dentro de `src`.

As predefinições disponíveis são:

- MongoDB (mongoose) - [Veja o guia](/discord/databases/mongodb)
- Firestore (typesaurus) [Veja o guia](/discord/databases/firestore)
- Firestore (firelord) [Veja o guia](/discord/databases/firestore)
- QuickDB (better-sqlite3)
